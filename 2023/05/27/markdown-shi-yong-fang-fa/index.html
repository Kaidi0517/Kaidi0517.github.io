簡要介紹

Markdown是一種輕量型標記語言，是一種語法. 以 .md 結尾的文本檔就是 Markdown 檔. 相較於Word，它更加像是HTML語言，並不是最淳樸的那種“所見即所得”. 它處處透露著一種極簡主義. 高效簡潔清晰的同時，又很簡單. 看起來舒服，語法簡單，尤其在處理純文本上有很大的優勢.

實際應用

所有要寫文字的時候都可以用上Markdown！

它可以讓你不再糾結什麼字體，什麼樣式，什麼排版. 而且邏輯清晰，層次分明.

可以用 Markdown 來寫各種筆記，演講稿，代碼的 README.md… .

工具

Markdown 只是一種語法. 那麼用來寫 Markdown 的文字編輯器可以使用VS Code

用法

1. 標題 [數個“#”+ 空格前置]

# 一級標題
## 二級標題
### 三級標題
#### 四級標題
##### 五級標題
###### 六級標題

標題會在目錄與大綱分級顯示，可以跳轉.

2. 粗體 [用“**”或“__”包圍]

**歡迎你!** 
__歡迎你!__

或者選取中想要強調的文字按下 Ctrl + B.

E.G.

歡迎你！

3. 斜體 [用“*”或“_”包圍]

*歡迎你* 
_歡迎你_

或者選取中想要強調的文字按下 Ctrl + I.

E.G.

歡迎你

(P.S. 斜體並強調 [用“***”或“___”包圍] ）

4. 刪除線 [用“~~”包圍]

~~我宣佈個事，我是個……!~~

E.G.

我宣佈個事，我是個……!

5. 代碼 [用“`”包圍]

`sudo rm -rf /*`

E.G.

sudo rm -rf/*（沒事別亂敲這個！）

6. 代碼塊 [按三個“’”並敲回車]

```
這裡就可以開始輸入你要的代碼了

print("Hello, World!")
``` // (這三個"`"一般都會補全，不用手動輸)

代碼塊裡可以選擇語言, 其會根據語言來自動高亮各個語句. 在選擇語言後, ``` 會變為 ```` + 對應語言.

7. 引用 [“>” + 空格 前置]

> 可是雪飄進雙眼
> > 看不見你橋牌的謊言

引用是可以嵌套的!

E.G.

可是雪飄進雙眼

看不見你橋牌的謊言

8. 無序列表 [“-“ 或 “+” + 空格 前置]

- 一顆是棗樹
+ 另一顆還是棗樹
* 還有一顆也是棗樹

三種前置符都可以, 敲回車會自動補全.

E.G.

  • 一顆是棗樹
  • 另一顆還是棗樹
  • 還有一顆也是棗樹

9. 有序列表 [數字 + “.” + 空格 前置]

我來這裡就為了三件事：
1. 公平
2. 公平
3. 還是**的公平！

敲回車會自動補全, 敲回車后按下 Tab 會縮進一級.

E.G.

我來這裡就為了三件事:

  1. 公平
  2. 公平
  3. 还是**的公平!

10. 鏈接 [用 “[ ]” + “(括號)” 包圍]

[來看看我贫瘠的博客罷](https://kaidi0517.github.io/)

E.G.

來看看我贫瘠的博客罷

11. 表格 [用 “|” 繪製表格邊框]

| 學號 |姓名| 年齡 |
| :--- | :---: | ---: | （引號的位置代表著左對齊，居中，右對齊）
|114514|劉像人|13|
|1919810|王慧好|16|

第一行為表頭, 並由第二行分割線決定對齊方式與長度, 第三行及之後即表格數據

E.G.

學號 姓名 年齡
114514 劉像人 13
1919810 王慧好 16

12. 圖片 [直接拖进來或者複製粘貼]

![圖片](圖片的位置)

13. 分割線 [按三個 “*” 或 “-“ 或 “_” 并敲回車]

***
--- 
___
（其實按三個及以上都可以）

由於 * 與 _ 均會自動補全, 所以我覺得 - 最為方便.

E.G.

14. Emoji表情

可以直接将 Emoji 表情複製进來

E.G.

😅🤤🤡

一個可以複製全Emoji的网站, 非常好用!

其他功能

其他功能要麼就用不到，要麼網站不支持，剩下的我就就不寫了。
